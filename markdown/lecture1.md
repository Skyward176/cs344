# Greedy Algorithms


A greedy algorithm has no foresight. It makes the best immediate choice with the hope that this converges to a global optimum.

For example, lets try to make an algorithm for scheduling.

We have list of events, *events*, where each event has a start and end time.

A greedy algorithm would 

schedule()
    pick soonest start time after last events end time
    set last last events end time to end time of new event


The problem is that although this can look optimal, it rarely ever actually works.
